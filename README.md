## Project Description
This project demonstrates JavaScript DOM Manipulation and Event Handling. It includes exercises on selecting HTML elements, traversing the DOM tree, modifying page content, creating and removing elements dynamically, handling user events, validating forms, and building a fully interactive To-Do List application.

The project was completed as part of the IYF Weekend Academy Season 11 Week 5 assignment.

## Technologies Used
- HTML5

- JavaScript
- Visual Studio Code

- GitHub

## Features
### Mini Project
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks
- Remaining task counter
- Clear completed tasks
- Edit tasks by double clicking

4. Launch `index.html` using Live Server
